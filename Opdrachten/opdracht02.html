2.1: No jQuery, why?

    Voordelen van jQuery
  • Animaties
  • Heel erg snel (Content Delivery Network aangepast)
  • Minder regels code
  • Werkt in crossbrowsers
  • Gemakkelijk te begrijpen en te gebruiken
  • Grote libary
  • Sterke open source community met veel plugins
  • Goed gedocumenteerd en veel tutorials
  • Het ondersteunt Ajax
    Nadelen van jQuery
  • Slechte performance doordat mensen het verkeerd gebruiken
  • Je leert geen Javascript schrijven en weet dus niet precies wat er gebeurt
  • Pagina kan breken door missende libaries
  • Je moet gehele libary inladen of delen daarvan
  • Batterij van smartphones gaan sneller leeg omdat hele libary open moet met libary calls
  • Functionaliteit kan gelimiteerd zijn
  • Soms moet je extra Javascript toevoegen om het te laten werken

2.2: Functionaliteiten jQuery

  1. Animatie
  2. Traversing door de DOM
  3. Events zijn gemakkelijk te gebruiken
  4. Ajax interactie met de server
  5. Cross-browser compatibel event handling
